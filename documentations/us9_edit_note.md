# User Story US9: Edit Note Modal

**Status:** To Do
**Description:** As a user, I want to be able to edit an existing note's name, so that I can keep my notes organized with relevant titles.

**Acceptance Criteria:**
- Create `EditNoteModal` component.
- Similar in functionality to `CreateNoteModal`.
- Pre-filled with the current note name.
- Updates the note name when "Save" is clicked.
- Includes unit tests verifying:
    - Rendering with current note name.
    - Saving with updated name.
    - Error handling.
