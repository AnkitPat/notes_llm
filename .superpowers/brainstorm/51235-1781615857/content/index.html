Refined Architecture: 1 Note → Many Resources

users

  • _id: ObjectId
  • email: string (unique)
  • isVerified: boolean
  • createdAt: Date
→
owns

notes

  • _id: ObjectId
  • userId: string (email)
  • name: string
  • resourceIds: Array<ObjectId>
  • updatedAt: Date
↓
owns multiple

resources

  • _id: ObjectId
  • noteId: ObjectId (ref)
  • name: string (Optional label?)
  • documents: Array<Doc>
  • text: Array<Text>
  • links: Array<Link>
